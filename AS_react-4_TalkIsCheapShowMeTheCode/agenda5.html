  • We need to import/export component.

    Two types of import/export

    1. default export - ONLY FOR 1 FILE.
      EG: export default Body(_Name of Component/Variable)
    2. default import - ONLY FOR 1 FILE.
      EG: import _componentName from _Path;
    3. Named export -> From single file, we need to export OR import multiple file.
      EG: export const LOGO_URL(_componentName).
    4. Named import -> You need to import it in {} Eg: import { CDN_URL } from _Path("../utils/constants");
