  • What is CDN?
    1. CDN : A Content Delivery Network (CDN) is geographically dispersed network of servers and data centers. The goal of CDN is to facilitate increased web performance and high availability by locating web content copies closer to end users.
    2. CDN servers store or cache copies of varies web content subsets from a host server, such as applications, video, audio, images and HTML files.
    3. CDN by bringing web content nearer to end-users, allows website publishers to control bandwidth consumption, reduce load times, and provide enhanced performance.
  • What is crossorigin?
    1. CORS - stands for cross-origin resource sharing. It helps in which one webpage requests for another domain for fetching out resources like audio, video, script etc. from third party server without leaking their credentials information.
    2. It defines a way of how a browser and server can interact to determine whether it is safe to allow the cross-origin request.
  • How browser understand Javascript? Javascript Virtual Machine.
    1. Browser like chrome decode Javascript using a compiler like V8 Engine.
    2. The browser has in-built Javascript engine that is responsible for interpreting and exeuting javascript code. When we visit web page that ocntains javascript, the browser downloads the Javascript code and passes it to the engine, which then reads the code and executes it.
